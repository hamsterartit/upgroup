исходники в папке src, билд в папке dist

в контактах нужно заменить ссылку на карту в iframe, так как текущая карта создана на мою почту

## Installation

```bash
  npm i
```
## Build dev

```bash
  npx gulp build --dev
```

## Build prod

```bash
  npx gulp build --prod
```

## Run dev

```bash
  npx gulp run --dev
```
